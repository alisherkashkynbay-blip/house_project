# Проект: Прогнозирование стоимости жилья

Этот проект выполнен в рамках курса  
**«Введение в анализ данных и машинное обучение»**.

Цель проекта — построить модель, которая по характеристикам объекта
недвижимости (площадь, комнаты, год постройки и т.д.)
прогнозирует его стоимость (задача регрессии).
